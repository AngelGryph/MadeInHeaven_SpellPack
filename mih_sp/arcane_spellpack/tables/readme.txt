These tables control the installation of various spell resources

items.2da

Installs simple items used by spells.  Needs the resref and name translation
string.


summons.2da: Simple summons for single spells.

Needs the cre resref, name translation string and animation to play on
summoning.  Installs the .cre file and the .eff file.  More complex summons,
or those used by multiple items and spells are in shared/summons instead.


master.2da: Master table that controls installing spells.

secondary.2da: Controls simple secondary spells that need installing.
