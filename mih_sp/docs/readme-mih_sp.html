Made in Heaven: Spell Pack

A Gibberlings Three Mod
Author: Angel

Version 7 - Check for the most recent version
Languages: English
Platforms: Windows, Mac OS X, and Linux

Overview

This is my spell pack for Baldur's Gate 1 and -2. It introduces several new arcane and divine spells to the game, as well as some new options for paladins.

Compatibility

This mod has been designed to work with BGEE, BG2EE and should be backwards compatible with BG2: ToB. The mega-mods EET and BGT are also supported. This mod is not compatible with older versions of the games.

As a WeiDU mod, this mod should be compatible with all other WeiDU mods. The core components have no known compatibility issues with any mods. The tweaks might conflict with other mods that do similar things, like Spell Revisions. As a spell pack, this mod should be installed fairly early, before any 'tweak' mods.

Installation

Before you install
If you have an older version of the MiH Spell Pack, be sure to completely remove it first before trying to install the new version!

Vanilla BG2 and Baldur's Gate Trilogy
THIS MOD IS DEVELOPED ON ENHANCED EDITION AND NOT FREQUENTLY TESTED ON THE OLD ENGINE
Before installing this mod, you must install the latest official patch, and you must also install the Throne of Bhaal Extender (ToBEx) and the G3 Fixpack version 11 or higher. This mod will not work correctly without those two installed!

BG1EE and Siege of Dragonspear
If you obtained your version of the game through Steam or GoG, you must either run a 'modmerge' or install 'A7-DlcMerger' before installing this mod (or any WeiDU-based mods, really). If you obtained your game from Beamdog, this is not required.

BG2EE and Extended Edition Trilogy
There are no requirements at this time. For EET, install this mod after EET has been installed but before the EET installation is finalized.

Windows
Made in Heaven: Spell Pack is packaged and installed with WeiDU and is distributed as a zip archive. Modern versions of Windows have support for zip files built in, just double-click the archive to open it, and extract the contents to your game file. If you are still using an older version of Windows, you should install WinZip or a similar program to extract the archive.

Your game folder should then contain a mih_sp folder and setup-mih_sp.exe. To install, double-click setup-mih_sp.exe and follow the instructions on screen.

Please run setup-mih_sp.exe in your game folder to reinstall, uninstall, or otherwise change components. To uninstall the entire mod without questions, run setup-mih_sp.exe --uninstall from a command line in your game directory.

Mac OS X
Made in Heaven: Spell Pack is packaged and installed with WeiDU and is distributed as a compressed archive. Extract the contents of the archive and move them to your game folder. Your game folder should then contain a mih_sp folder, setup-mih_sp, and setup-mih_sp.command. To install, simply double-click setup-mih_sp.command and follow the instructions on screen.

Linux
Extract the contents of the mod archive to your game installation directory.

If you haven't already, download the latest version of WeiDU for Linux from weidu.org and extract (at least) the weidu executable to a $PATH directory (e.g. /usr/bin).

To install, open a terminal, cd to your game installation directory and run weidu --log setup-mih_sp.debug mih_sp/setup-mih_sp.tp2 from your game installation directory to install or uninstall the mod.

Note: The readme files of some older mods recommend running tolower on Linux prior to installation. Don't do this on an Enhanced Edition game, you will break it! Modern versions of tolower will actually refuse to run in a EE game directory. At any rate, the contents of MiH: Spell Pack are already all lower case, there is no need for tolower.

New Game required!
After installing this mod you must start a new game. This mod edits area files and stores, both of which are stored in your save game after you have visited them. Continuing a saved game after installing this mod will have unexpected and likely undesired results.

Component Descriptions

Core Components

This mod consists of several components, each of which can be installed or removed independently from the others.

Made in Heaven: Arcane Spellpack
This installs new spells for wizards and bards. The spells will be available on the selection screen when creating a new character, and spell scrolls are distributed throughout the environment.

Arcane Spell List

Note: This component makes several changes to better match the game to known Forgotten Realms lore. This can be turned off in the ini file if you really want to, but some things will then look strange.

Note: Ninth level spell slots are actually fairly limited because the game also puts HLAs and a bunch of BS carried over from TotSC there. This may make some spells unlearnable by sorcerers, although wizards can learn it from scrolls just fine. I have experimental code to free up more slots, but it's disabled by default. Check the ini file if you want to play with it.

Icewind Dale: Arcane Spellpack (CamDawg/DavidW)

This installs arcane spells from Icewind Dale into BG1/BG2. This uses the IWDspells libary provided by CamDawg and DavidW, and as such is exactly the same component as found in CamDawg's IWDificiation and DavidW's Sword Coast Stratagems. If Icewind Dale arcane spells from either of these are already installed, this component will do nothing other than print a message.

Made in Heaven: Divine Spellpack
This installs new spells for clerics and druids. The new spells will be added to the spell list as the characters level up, just like other spells for these classes.

Divine Spell List

Icewind Dale: Divine Spellpack (CamDawg/DavidW)

This installs divine spells from Icewind Dale into BG1/BG2. This uses the IWDspells libary provided by CamDawg and DavidW, and as such is exactly the same component as found in CamDawg's IWDificiation and DavidW's Sword Coast Stratagems. If Icewind Dale arcane spells from either of these are already installed, this component will do nothing other than print a message.

Improved Paladin Powers
This component revises and extends paladin powers as follows:

  • Detect Evil is now usable at will.
  • Protection from Evil only affects the paladin and can be used only once per day, but has a greatly extended duration.
  • All good paladins are immune to disease.
  • All good paladins can cast Divine Favor once per day at level 3. This extends to twice per day at level 10 and three times per day at level 17. Divine Favor gives the paladin a small bonus to hit and damage.
  • True paladins, cavaliers and inquisitors can Smite Evil once per day. Undead hunters can Smite Undead instead, and blackguards can Smite Good. The smite deals 1d6 magical damage for every 3 paladin levels.
  • True paladins and cavaliers gain an extra use of Lay on Hands every 7th level (max. 5 at level 28).
  • True paladins and cavaliers can Cure Disease once per day starting at 2nd level. At 6th level they can do this twice per day.
  • Paladins can start casting divine spells at 4th level.
  • Paladins can opt to gain an extra use of Lay on Hands or Smite Good/Evil/Undead as a high-level ability, up to three times for each.
  • The Cavalier bonus against dragons has been extended to wyverns.
  • Paladins can achieve mastery (three stars) in "knightly" weapons like longswords, bastard swords, two-handed swords, flails, maces, warhammers and halberds.

Improved Ranger Powers
This component revises and extends bard powers as follows:

  • The use of Charm Animal (for all except Archers) is limited to three times per day at level 5, but the target suffers a -1 penalty on saves for every odd ranger level up to -10 at level 21.
  • Rangers can start casting divine spells at 4th level.
  • Rangers and Stalkers can achieve mastery (three stars) in weapons typically used for dual-wielding like daggers, shortswords, and scimitars, and in all types of bows. Beast Masters can only gain mastery in all kinds of bows. The weapon proficiencies for Archers remain unchanged.

Improved Bard Powers
This component revises and extends bard powers as follows:

  • Bards can cast Friends once per day.
  • Bards add the following spells to their spell list automatically: Cure Light Wounds, Slow Poison, Cure Disease, Cure Serious Wounds and Cure Critical Wounds.

Bhaalspawn Powers Pack (EXPERIMENTAL)
This component enhances and changes the bhaalspawn powers gained through the course of Baldur's Gate 1. This is a work in progress and liable to change. Currently only the 'good' powers are affected.

Epic Spell Pack (BETA)
This component installs a few new high-level spell options for both arcane- and divine casters. This is still in development and may change in the future.

Spell Effect Tweaks

Revised Dispel Magic (EXPERIMENTAL)
Due to the way the 2nd edition rules work, Dispel/Remove Magic are almost useless to players and nearly overpowered for enemies, which are almost always higher level than your casters. This component changes these spells so that their success flat-out depends on caster level, not on level difference. The chance to succesfully dispel is 50%, plus 2% per caster level. This is an experimental component still being tested for game balance and may change.

True Meteor Swarm (EXPERIMENTAL)
This tweak will introduce the true Meteor Swarm spell as described in the 2e Player's Handbook; four 10d4 fireballs the caster can launch at up to four targets. The existing Meteor Swarm spell will be renamed to Rain of Destruction. The new spell scroll is available from the Arcana Archives. Enemies will keep using the old spell, and it will remain available to players too. This is still a work in progress and may change.

Prevent Damage during Time Stop
This tweak makes it impossible to deal damage to time-stopped creatures, they will be 100% resistant to everything. This is mostly to prevent the incredibly cheap Time Stop melee tactic.

Remove All Time Stop Immunity
This tweak will remove immunity to Time Stop from all creatures. Casters remain of course immune to their own Time Stop spells.

Spell Improvements
This is a collection of minor tweaks to various spells.

  • Protection from Normal Missiles: In addition to its usual effects, this spell will now reduce missile damage to the recipient by 50%.
  • Wyvern Call: The wyvern called will now be an 11 HD greater wyvern similar to those encountered in Durlag's Tower. Very high-level casters will call up to three wyverns.
  • Shillelagh: The Shillelagh is treated as a magical club with a +1 enhancement bonus and no penalty for non-proficiency.
  • Barkskin: If the recipient already has a better armor class than what the spell gives, they will still get an AC bonus of +1 to +5.
  • Flame Blade: The flaming sword is treated as a magical scimitar with no penalty for non-proficiency. It can strike any target not immune to fire with a +4 to hit and deals 1d8 points of fire damage, +1 for every two levels. Strength does not apply.

Revised Spell Learning XP
This small component revises the experience gained from writing new spells to the spellbook to more sane values, making it actually worth doing in BG1 while toning down the ridiculous values in BG2. This is similar to the component from BGT Tweaks. The XP gains are as follows: 50/100/200/500/1000/2000/3000/4000/5000.

Spell Availability Tweaks

Expanded Temple Services
This component expands the services of all temples in the game. You can now purchase the spells Cure Disease and Restoration at most temples. If you installed Spell Restorations from MiH F&R, Cure Blindness/Deafness will be available as well, and if you installed the Divine Spellpack from this mod, Break Enchantment will also be added.

Sixth Circle Spell Scrolls in Siege of Dragonspear
Five sixths circle spells were actually in Tales of the Sword Coast, although not usable by the player: Invisible Stalker, Globe of Invulnerability, Tenser's Transformation, Flesh to Stone and Death Spell. Now that Siege of Dragonspear extends the story, it seems only fair to let the player actually have these spells. This will be most helpful if you have a mod that removes the XP cap so your wizard actually can reach level twelve and have a sixth circle spell slot, but even if not they can still use the scrolls. One copy of each is available at the Sorcerous Sundries (only after SoD starts), and at quartermaster Belegarm in the camp.

Expert Configuration Options

Within the mod folder, there is a file called mih_sp.ini. This file can be used to fine-tune certain aspects of this mod. It is meant as an expert option; for normal installation you can leave this file as it is. If you do edit this file, you must reinstall the mod, or at least the components for which you changed settings.

Which spells get installed by the spell packs is determined by the file tables/spells.2da in the subdir folders of the components. It is possible to add your own spells here. The spell file goes in the subdir spells and the icons go into graphics. There are other similar tables that control the installation of things like summoned creatures and simple items like paws. You can also delete a spell you really don't like here, but this may have unexpected results down the line so be careful.
